# Kangra Temples | Discover Top Destinations

A premium, highly animated web application designed to showcase the spiritual and cultural heritage of the Kangra district in Himachal Pradesh, India. This project provides a seamless user experience for tourists and devotees looking to explore famous temples like Jalamukhi, Baijnath, and Chamunda Devi.

---

## ✨ Features

*   **Premium Visual Experience**: Modern design with glassmorphism, smooth gradients, and rich micro-animations.
*   **Dynamic Image Slider**: Responsive homepage carousel showcasing the beauty of the Himalayas and the temples.
*   **Temple Gallery**: A responsive grid layout featuring various temples with staggered entry animations.
*   **Detailed Information**: Individual pages for each temple including travel guides and integrated Google Maps.
*   **Contact System**: A functional contact form with simulated success feedback for improved user interaction.
*   **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.

---

## 🛠️ Technology Stack

*   **Frontend**: HTML5, Vanilla CSS3 (Custom Animations & Keyframes), ES6+ JavaScript.
*   **Backend**: Node.js, Express.js.
*   **Utilities**: Font Awesome (Planned/Optional), SendGrid (API Integration Ready).

## 🚀 Getting Started

Follow these steps to run the project on your local machine.

### Prerequisites
*   [Node.js](https://nodejs.org/) (v14 or higher recommended)
*   [Git](https://git-scm.com/)

### Installation
1. Clone the repository or download the source code.
2. Open terminal in the project directory.
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
To start the development server, run:
```bash
npm start
```
The server will start at `http://localhost:3000`. Open your browser and navigate to this address.

## 📂 Project Structure

*   `index.html`: The main landing page.
*   `temples.html`: The full gallery of temples.
*   `about.html`: Detailed information about the Kangra region.
*   `contact.html`: Contact form with simulated submission.
*   `style.css`: The core design system and animation logic.
*   `server.js`: Express server to handle routing and API requests.
*   `slider.js`: Logic for the homepage image carousel.
